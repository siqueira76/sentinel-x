# 📊 Sistema 2: Sentinel-Core (Backend & Central de Comando)

## 1. Função Estratégica
Este sistema centraliza as informações de todos os Agents espalhados pela região. Ele é o repositório da verdade e o motor de regras de segurança.

## 2. Componentes de Armazenamento
- **TimescaleDB (PostgreSQL):** Gerencia milhões de registros de fluxo, permitindo consultas instantâneas de "fluxo por minuto" e "mapas de calor".
- **Evidence Storage (S3):** Armazena o `crop` (foto recortada) da placa com metadados embutidos, garantindo prova visual para segurança pública.

## 3. Regras de Negócio e Alertas
- **Motor de Whitelist/Blacklist:** Comparação instantânea com placas de veículos roubados ou de interesse administrativo.
- **Webhooks de Integração:** Capacidade de enviar alertas instantâneos para Telegram, WhatsApp ou Centrais de Polícia.

## 4. API de Consumo
Exposição de endpoints REST seguros para que o Dashboard e o MCP Server possam consultar a base de dados histórica.
