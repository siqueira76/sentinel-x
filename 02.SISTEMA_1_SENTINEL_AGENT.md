# ⚙️ Sistema 1: Sentinel-Agent (Edge Computing)

## 1. Função Estratégica
O Agent é o software instalado no **Ubuntu Server** local (ex: em Monte Gordo ou Praia do Forte). Sua função é evitar o tráfego de vídeo pesado pela rede, processando tudo na "borda" (Edge) e enviando apenas metadados leves para a nuvem.

## 2. Detalhes Técnicos (Stack Java)
- **Engine de Captura:** Utiliza `JavaCV` com decodificação via hardware (Intel QuickSync/OpenVINO) para minimizar o uso de CPU.
- **Pipeline de IA:**
    - **Detector:** YOLOv10 para segmentação de veículos, motos e pedestres.
    - **LPR Engine:** Algoritmo customizado para placas Mercosul (leitura de caracteres sobre fundo azul/branco).
    - **Tracking:** Algoritmo de fluxo para evitar contagem duplicada do mesmo veículo.

## 3. Especificações da Câmera (Input)
- **Modelo Homologado:** Xelpon XSVB 101.
- **Configuração:** Stream RTSP Principal (3MP) para LPR e Sub-stream (D1) para detecção de pedestres.

## 4. Diferencial Business
- **Autonomia Offline:** Caso a internet caia, o Agent armazena as capturas localmente em um banco SQLite e sincroniza automaticamente ao retornar a conexão.
